# Mathematical Modeling using Wolfram Mathematica

This repository contains my academic project on mathematical modeling and visualization using Wolfram Mathematica and LaTeX.

## Project Overview
The project focuses on developing mathematical models using symbolic and numerical methods. It demonstrates the application of calculus and algebra to analyze real-world and theoretical problems.

## Key Features
- Developed mathematical models using symbolic and numerical computation
- Created 2D and 3D visualizations
- Applied concepts from calculus and algebra
- Documented results using LaTeX

## Tools Used
- Wolfram Mathematica
